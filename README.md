# The Chronicles of UR Vinis

#### The Game

It runs in a terminal. It has an open world. It is (re)written in [LiveScript](http://livescript.net/). Beware of the red wolves: they will track you. Search the words of wisdom.

## Running

```bash
npm install
make run
```
